# Cervical Cancer Risk Prediction

A machine learning project focused on predicting cervical cancer risk factors using clinical and demographic data.

## Overview

This project explores various machine learning approaches to identify and predict risk factors associated with cervical cancer. The analysis leverages the Kaggle cervical cancer risk factors dataset to develop predictive models.

## Experimentation

The project follows an iterative experimentation approach:

- **Data Exploration**: Initial analysis of the dataset to understand risk factors and data distributions
- **Feature Engineering**: Creation and selection of relevant features from raw data
- **Model Development**: Implementation and comparison of multiple ML algorithms
- **Hyperparameter Tuning**: Optimization of model parameters for improved performance
- **Validation**: Cross-validation and testing to ensure model generalization

Experiments are tracked in Jupyter notebooks (`test.ipynb`, `test1.ipynb`) to document findings and iterative improvements.

## Getting Started

### Prerequisites

- Python 3.x
- Virtual environment (recommended)

### Installation

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Open notebooks to explore experiments:
```bash
jupyter notebook notebook/test.ipynb
```

## Usage

Run the Jupyter notebooks to reproduce experiments and analysis. Each notebook contains detailed comments and visualizations of the experimentation process.
